# computer-store-sales-dashboard
An interactive Excel dashboard for analyzing computer store sales data using Pivot Tables, Pivot Charts, and Data Visualization.

# Project Overview
This project analyzes computer store sales data using Microsoft Excel.

# The dashboard provides insights into:
- Product-wise Sales
- Brand Performance
- City-wise Revenue
- Sales Analysis

# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

# Steps Followed
1. Collected the sales dataset.
2. Cleaned and organized the data.
3. Created Pivot Tables.
4. Generated Pivot Charts.
5. Applied Conditional Formatting.
6. Designed an interactive dashboard.
7. Analyzed sales performance and generated insights.

# Key Insights
- Identified top-performing products.
- Compared sales across different cities.
- Analyzed brand performance.
- Visualized sales data through charts and dashboards.

# Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Data Visualization
